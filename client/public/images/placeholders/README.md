# Image Placeholders

Thư mục này chứa các ảnh placeholder được sử dụng khi không có ảnh thực từ database.

## Danh sách placeholder files cần có:

1. **tour-placeholder.jpg** - Ảnh mặc định cho tour (400x300)
2. **place-placeholder.jpg** - Ảnh mặc định cho địa điểm (400x300)
3. **avatar-placeholder.jpg** - Ảnh đại diện mặc định (150x150)
4. **cover-placeholder.jpg** - Ảnh bìa mặc định (1200x400)
5. **3d-model-placeholder.jpg** - Placeholder cho 3D models (400x300)
6. **3d-model-preview.png** - Preview image cho 3D model (400x400)
7. **video-thumbnail.jpg** - Thumbnail cho video (320x180)
8. **gallery-1.jpg ~ gallery-6.jpg** - Ảnh gallery cho Blog (400x300 mỗi ảnh)
9. **feature-place-1.jpg ~ feature-place-3.jpg** - Feature images cho Places page
10. **scenic-placeholder.jpg** - Ảnh scenic view (600x400)
11. **auth-bg.jpg** - Background cho Auth page (1920x1080)
12. **cta-bg-placeholder.jpg** - Background cho CTA section (1200x600)

## Temporary Usage

Hiện tại có file `placeholder.svg` để dùng tạm. Khi có ảnh thực, hãy thêm vào thư mục này với tên file tương ứng.

Có thể dùng placeholder online như:

- https://placehold.co/400x300/e2e8f0/64748b?text=Tour
- https://placehold.co/150x150/e2e8f0/64748b?text=Avatar
